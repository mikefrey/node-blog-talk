#### Generate the app

```bash
$ express --ejs --css stylus myapp
```


#### Install dependencies

```bash
$ cd myapp
$ npm install
```


#### Test the app

```bash
$ node app.js
```

Go to http://localhost:3000


#### Install leveldb dependencies

```bash
$ npm install level --save
$ npm install level-sublevel --save
```


#### Create models dir and data dir
```bash
$ mkdir models
$ mkdir data
```
